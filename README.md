# CryptechTest Game

CryptechTest is an experimental gamemode of survival multiplayer that challenges players in a unique and dynamic environment, emphasizing teamwork and strategic thinking. This gamemode introduces a variety of mods that expand gameplay mechanics and create diverse enviroments for players to explore and conquer.

## Overview

The CryptechTest gamemode has an emphasis on space travel, construction, machines and automations.
